5. Listák

A listák olyan adattípusok, amelyek több elemet képesek tárolni egyetlen változóban. A lista elemei különböző típusúak lehetnek, és a lista hossza dinamikusan változhat. A listák indexelése 0-tól kezdődik, és a negatív indexek a lista végétől számítanak visszafelé.

        gyumolcsok = ["alma", "banán", "cseresznye"]
        print(gyumolcsok[0])  # Kimenet: alma
        gyumolcsok.append("datolya")
        print(gyumolcsok)  # Kimenet: ['alma', 'banán', 'cseresznye', 'datolya']

Ebben a példában létrehoztunk egy 'gyumolcsok' nevű listát, amely három gyümölcs nevét tartalmazza. Az első elem eléréséhez az indexelést használtuk, majd a append() metódussal hozzáadtunk egy új elemet a lista végéhez.

6. Szótárak (Dictionary)

        szemely = {"név": "János", "kor": 25, "város": "Budapest"}
        print(szemely["név"])  # Kimenet: János
        szemely["kor"] = 26
        print(szemely)  # Kimenet: {'név': 'János', 'kor': 26, 'város': 'Budapest'}

9. Listák szűrése és műveletek (List Comprehensions)

        szamok = [1, 2, 3, 4, 5, 6, 7, 8, 9]
        paros_szamok = [szam for szam in szamok if szam % 2 == 0]
        print(paros_szamok)  # Kimenet: [2, 4, 6, 8]

10. Hibakezelés (Exception Handling)

        try:
            szam = int(input("Adj meg egy számot: "))
            print(10 / szam)
        except ZeroDivisionError:
            print("Hiba: osztás nullával!")
        except ValueError:
            print("Hiba: nem számot adtál meg!")

11. Osztályok és objektumok (Class and Objects)

        class Szemely:
            def __init__(self, nev, kor):
                self.nev = nev
                self.kor = kor

            def bemutatkozas(self):
                print(f"Hello, én {self.nev} vagyok és {self.kor} éves vagyok.")

        szemely1 = Szemely("Anna", 30)
        szemely1.bemutatkozas()  # Kimenet: Hello, én Anna vagyok és 30 éves vagyok.
