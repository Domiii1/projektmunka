Python Programozási Alapok

1. Változók és kiíratás

name = "Tanuló" print("Helló,", name)

2. Feltételes elágazás

age = 18 if age >= 18: print("Felnőtt vagy.") else: print("Még nem vagy felnőtt.")

3. Ciklusok

for i in range(5): print("Szám:", i)

4. Függvények

def osszeg(a, b): return a + b print(osszeg(3, 5))

5. Listák

A listák olyan adattípusok, amelyek több elemet képesek tárolni egyetlen változóban. A lista elemei különböző típusúak lehetnek, és a lista hossza dinamikusan változhat. A listák indexelése 0-tól kezdődik, és a negatív indexek a lista végétől számítanak visszafelé.

gyumolcsok = ["alma", "banán", "cseresznye"] print(gyumolcsok[0]) # Kimenet: alma gyumolcsok.append("datolya") print(gyumolcsok) # Kimenet: ['alma', 'banán', 'cseresznye', 'datolya']

Ebben a példában létrehoztunk egy 'gyumolcsok' nevű listát, amely három gyümölcs nevét tartalmazza. Az első elem eléréséhez az indexelést használtuk, majd a append() metódussal hozzáadtunk egy új elemet a lista végéhez.

6. Szótárak (Dictionary)

szemely = {"név": "János", "kor": 25, "város": "Budapest"} print(szemely["név"]) # Kimenet: János szemely["kor"] = 26 print(szemely) # Kimenet: {'név': 'János', 'kor': 26, 'város': 'Budapest'}

7. Szekvenciák (Stringek)

szoveg = "Helló világ!" print(szoveg.lower()) # Kimenet: helló világ! print(szoveg.upper()) # Kimenet: HELLÓ VILÁG!

8. Listák szűrése és műveletek (List Comprehensions)

szamok = [1, 2, 3, 4, 5, 6, 7, 8, 9] paros_szamok = [szam for szam in szamok if szam % 2 == 0] print(paros_szamok) # Kimenet: [2, 4, 6, 8]

9. Osztályok és objektumok (Class and Objects)

class Szemely: def __init__(self, nev, kor): self.nev = nev self.kor = kor def bemutatkozas(self): print(f"Hello, én {self.nev} vagyok és {self.kor} éves vagyok.") szemely1 = Szemely("Anna", 30) szemely1.bemutatkozas() # Kimenet: Hello, én Anna vagyok és 30 éves vagyok.

10. Lambda függvények

negyzet = lambda x: x ** 2 print(negyzet(4)) # Kimenet: 16

11. Modulok importálása

import math print(math.sqrt(16)) # Kimenet: 4.0
