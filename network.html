Kapcsoló és Router Alapkonfiguráció

1. Kapcsoló alapvető beállításai

        enable
        configure terminal
        hostname Switch1
        interface vlan 1
        ip address 192.168.1.1 255.255.255.0
        no shutdown
        exit

2. Router alapszintű konfiguráció

        enable
        configure terminal
        hostname Router1
        interface GigabitEthernet0/0
        ip address 192.168.1.254 255.255.255.0
        no shutdown
        exit

3. Haladó hálózati beállítások

Statikus útvonal beállítása:

        ip route 192.168.2.0 255.255.255.0 192.168.1.2

DHCP konfiguráció egy routeren:

        ip dhcp pool LAN
        network 192.168.1.0 255.255.255.0
        default-router 192.168.1.254
        dns-server 8.8.8.8
        exit
