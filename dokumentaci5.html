1. Hálózati Tervezés

A kis- és közepes vállalatok számára fontos a megbízható és biztonságos hálózati infrastruktúra. Az IPv4 címzés lehetővé teszi az eszközök egyértelmű azonosítását.

1.1 Címzési struktúra

Ajánlott címzési tartomány:

192.168.0.0/24

Ez 254 eszközt képes kiszolgálni, alhálózatokra osztható igény szerint.

2. Egytáblás Adatbázis

A vállalati munkatársak adatainak tárolására egy egyszerű SQL-alapú adatbázis javasolt.

2.1 Tábla szerkezete

  • ID: Egyedi azonosító
  • Vezetéknév
  • Keresztnév
  • Email
  • Telefonszám

2.2 SQL létrehozás

CREATE TABLE Munkavallalok (
  id INT PRIMARY KEY AUTO_INCREMENT,
  vezeteknev VARCHAR(50),
  keresztnev VARCHAR(50),
  email VARCHAR(100),
  telefonszam VARCHAR(20)
);

2.3 Műveletek

-- Beszúrás
INSERT INTO Munkavallalok (vezeteknev, keresztnev, email, telefonszam)
VALUES ('Kovács', 'Anna', 'anna.kovacs@example.com', '06201234567');

-- Módosítás
UPDATE Munkavallalok
SET email = 'anna.uj@example.com'
WHERE id = 1;

-- Lekérdezés
SELECT * FROM Munkavallalok WHERE id = 1;
