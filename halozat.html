Hálózati Alapok

A hálózatok lehetővé teszik az eszközök közötti kommunikációt. A helyi hálózat (LAN) és a globális hálózat (WAN) alapvető fogalmak, amelyeket minden informatikusnak ismernie kell.

  • IP-cím: Azonosítja az eszközt a hálózaton belül.
  • MAC-cím: Fizikai azonosító, amely az eszköz hálózati interfészéhez tartozik.
  • Router: Összeköti a különböző hálózatokat, irányítja az adatforgalmat.
  • Switch: Az eszközök közötti adatátvitelt kezeli a LAN-on belül.

Python Alapok

A Python egy könnyen tanulható, sokoldalú programozási nyelv, amelyet gyakran használnak hálózati automatizálásra is.

Egyszerű példa: IP-cím kiírása

ip = "192.168.1.1"
print("Az IP-cím:", ip)

Lista bejárása

ip_lista = ["192.168.1.1", "10.0.0.1", "172.16.0.1"]
for ip in ip_lista:
    print("Cím:", ip)

Függvény használata

def ping_cim(cim):
    print(f"{cim} pingelve...")

ping_cim("192.168.1.1")
