Kapcsoló és Router Alapkonfiguráció

1. Alapvető beállítások kapcsolón

Íme néhány alapvető parancs egy Cisco kapcsoló konfigurálásához:

        enable
        configure terminal
        hostname Switch1
        interface vlan 1
        ip address 192.168.1.1 255.255.255.0
        no shutdown
        exit

2. Router alapszintű konfiguráció

Router beállítása egy egyszerű LAN számára:

        enable
        configure terminal
        hostname Router1
        interface GigabitEthernet0/0
        ip address 192.168.1.254 255.255.255.0
        no shutdown
        exit

3. Haladó hálózati beállítások

Statikus útvonal beállítása:

        ip route 192.168.2.0 255.255.255.0 192.168.1.2

DHCP konfiguráció egy routeren:

        ip dhcp pool LAN
        network 192.168.1.0 255.255.255.0
        default-router 192.168.1.254
        dns-server 8.8.8.8
        exit

Python Nyelv Alapjai

1. Változók és kiíratás

Egyszerű Python kód:

        name = "Tanuló"
        print("Helló,", name)

2. Feltételes elágazás

Hogyan működik az if-else szerkezet Pythonban:

        age = 18
        if age >= 18:
            print("Felnőtt vagy.")
        else:
            print("Még nem vagy felnőtt.")

3. Ciklusok

For ciklus használata:

        for i in range(5):
            print("Szám:", i)

4. Függvények

Egyszerű Python függvény:

        def osszeg(a, b):
            return a + b
        
        print(osszeg(3, 5))
