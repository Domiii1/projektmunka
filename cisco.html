Vállalati Hálózati Topológia

Hálózati Eszközök és IP-címek

Ez a hálózati topológia egy egyszerű vállalati hálózatot mutat be, amely tartalmazza a routert, switch-et, PC-ket és egy szervert.

Hálózati Eszközök:

Eszköz IP-cím Leírás
Router 192.168.1.1 Hálózati forgalom irányítása
Switch N/A Hálózati eszközök összekapcsolása
PC1 192.168.1.10 Felhasználói számítógép
PC2 192.168.1.20 Felhasználói számítógép
PC3 192.168.1.30 Felhasználói számítógép
Szerver 192.168.1.50 Web és fájl szerver

Hálózati Diagram

Hálózati Beállítások

A routeren a következő beállítások szerepelnek:

Router> enable
Router# config terminal
Router(config)# interface g0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown

A PC-k IP-cím beállítása:

PC1: 192.168.1.10
PC2: 192.168.1.20
PC3: 192.168.1.30
Szerver: 192.168.1.50

Tesztelés

Ping teszt a hálózat tesztelésére:

C:\> ping 192.168.1.1 (router IP)
C:\> ping 192.168.1.50 (szerver IP)
