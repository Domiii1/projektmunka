Bevezetés

A hálózati eszközök, mint a kapcsolók (switch) és routerek alapvető szerepet játszanak a számítógépes hálózatokban. A kapcsolók lehetővé teszik az eszközök közötti adatátvitelt a helyi hálózatban, míg a routerek a különböző hálózatok közötti adatforgalmat irányítják. Az alábbi dokumentáció célja, hogy bemutassa a kapcsolók és routerek alapvető konfigurálását és azok szerepét egy hálózati környezetben.

1. Kapcsoló (Switch) Alapkonfiguráció

A kapcsolók feladata, hogy az eszközök között adatokat továbbítsanak ugyanabban a helyi hálózatban (LAN). A kapcsolók a MAC-címek alapján irányítják az adatcsomagokat a megfelelő portok között. Az alapkonfiguráció célja, hogy a kapcsoló működését gyorsan és hatékonyan beállítsuk.

1.1 Kapcsoló alapértelmezett konfigurációja

Amikor egy kapcsolót először bekapcsolunk, az alapértelmezett konfigurációt kell beállítanunk. A legfontosabb feladatok:

  • A kapcsoló neve
  • A soros port beállításai (ha szükséges)
  • Az eszköz védelme (jelszó beállítása)
  • A VLAN-ok konfigurálása (ha szükséges)

1.2 Alap konfigurációs lépések Cisco kapcsolókon

Switch> enable
Switch# configure terminal
Switch(config)# hostname Kapcsolo1
Switch(config)# enable secret jelszo123
Switch(config)# vlan 10
Switch(config-vlan)# name Sales
Switch(config-vlan)# exit
Switch(config)# interface gigabitethernet 0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config)# end
Switch# write memory

1.3 Kapcsoló konfigurációs tesztelés

A konfiguráció ellenőrzésére a következő parancsokat használhatjuk:

  • show running-config: A kapcsoló jelenlegi konfigurációjának megtekintése
  • show vlan brief: A VLAN-ok listájának megjelenítése
  • show interface status: A portok állapotának ellenőrzése

2. Router Alapkonfiguráció

A routerek a különböző hálózatok közötti adatforgalmat irányítják. A routerek beállítása alapvetően különbözik a kapcsolókétól, mivel a routerek IP-címekkel és alhálózati maszkokkal dolgoznak.

2.1 Router alapértelmezett konfigurációja

A routerek beállítása során a legfontosabb lépések a következőket tartalmazzák:

  • A router neve
  • Az interfészek konfigurálása
  • Az IP-címek és alhálózati maszkok hozzárendelése
  • A routing protokollok beállítása

2.2 Alap konfigurációs lépések Cisco routereken

Router> enable
Router# configure terminal
Router1(config)# hostname Router1
Router1(config)# interface gigabitethernet 0/0
Router1(config-if)# ip address 192.168.1.1 255.255.255.0
Router1(config-if)# no shutdown
Router1(config)# router rip
Router1(config-router)# version 2
Router1(config-router)# network 192.168.1.0
Router1(config)# end
Router1# write memory
